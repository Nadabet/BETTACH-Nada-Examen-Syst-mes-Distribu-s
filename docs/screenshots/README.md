# Screenshots Directory

This directory contains screenshots for the README documentation.

## Required Screenshots

1. **eureka-dashboard.png** - Eureka Server dashboard showing all registered services
2. **keynote-swagger.png** - Keynote Service Swagger UI
3. **conference-swagger.png** - Conference Service Swagger UI  
4. **angular-conferences.png** - Angular app conferences list view
5. **angular-keynotes.png** - Angular app keynotes list view
6. **angular-reviews.png** - Angular app reviews page
7. **keycloak-console.png** - Keycloak admin console
8. **gateway-routes.png** - Gateway routes configuration

## How to Generate Screenshots

1. Start all services using Docker Compose or manually
2. Navigate to each URL listed below
3. Take screenshots and save them in this directory

### URLs to Capture

- Eureka: http://localhost:8761
- Keynote Swagger: http://localhost:8081/swagger-ui.html
- Conference Swagger: http://localhost:8082/swagger-ui.html
- Angular App: http://localhost:4200
- Keycloak: http://localhost:8080
- Gateway Actuator: http://localhost:9999/actuator/gateway/routes
